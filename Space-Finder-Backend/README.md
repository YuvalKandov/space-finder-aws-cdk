# Space Finder Backend

This folder contains the backend infrastructure and services for Space Finder.

## Technologies
- AWS CDK (TypeScript)
- Lambda
- API Gateway
- DynamoDB
- S3
- Cognito
- CloudWatch & SNS
- Jest for testing

## Key Concepts Demonstrated
- Infrastructure as Code
- JWT authentication
- IAM role-based permissions
- Serverless monitoring
- AWS SDK mocking in tests

## Deployment
npm install
npm run deploy

Environment variables required:
- SLACK_WEBHOOK_URL
